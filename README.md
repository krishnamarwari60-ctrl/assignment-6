## Assignment  DATA ANALYSIS WITH SQL


## Description
This tasks performed to modify a database table structure, populate newly added columns, and delete records based on specific conditions. These operations help maintain data accuracy, track record updates, and remove outdated or inactive data.



##  Task 1: Modify Table Structure
# Add and Populate New Columns
    *  Description
  The table structure is updated to include additional columns that support better record tracking and status management.
# Actions Performed
Added a LastUpdated column to store the date when a record was last modified.
Added a Status column to indicate whether a record is active or inactive.


## Task 2: Delete Records Based on Conditions
 *  Description
      Unnecessary or outdated records are removed from the table using defined conditions to keep the dataset clean and relevant.
# Deletion Criteria
Records with Status = 'Inactive'.
Records where LastUpdated is older than a specified date.
